# DNSMate

Minimal one-click DNS changer for Windows 10 & 11 preloaded with common DNS providers for gaming and daily use. No installation, no local file storage, no unnecessary downloads, no data collection, just what you'd expect it to be.

## Usage

Run the following command in PowerShell:

```ps1
irm "https://xeptore.dev/dnsmate" | iex
```
